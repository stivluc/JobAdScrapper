Éditeur de configuration
Aide et documentation
Profil utilisateur
  • skills: Compétences séparées par des virgules
  • experience_years: Nombre d'années d'expérience
  • education_level: Niveau d'éducation
Critères de recherche
  • keywords: Liste des types de postes
  • locations: Liste des localisations
  • salary_min/max: Fourchette salariale
  • remote_ok: Télétravail accepté (true/false)
Paramètres scraper
  • max_google_queries: Nombre de requêtes Google
  • max_jobs_total: Nombre max d'offres
  • delay_between_queries: Délai anti-détection
  • headless: Mode invisible (true/false)
Attention: Une configuration incorrecte peut empêcher le scraper de fonctionner. Assurez-vous de respecter la syntaxe YAML et validez vos modifications.
Template de configuration

Voici un template de base pour la configuration :

# Configuration du scraper d'emploi
user_profile:
  skills: "Python, JavaScript, React, Node.js, SQL"
  experience_years: 5
  education_level: "Master"

search_criteria:
  keywords: ["développeur", "ingénieur logiciel", "full stack"]
  locations: ["Paris, France", "Lyon, France", "télétravail"]
  salary_min: 45000
  salary_max: 80000
  remote_ok: true

scraper_settings:
  max_google_queries: 15
  max_jobs_total: 100
  delay_between_queries: 8
  headless: false
